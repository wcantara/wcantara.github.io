RiboCAT

RiboCAT (Ribonucleotide Chemical probing Analysis Tool) is a tool for analyzing capillary electrophoresis data from SHAPE (Selective 2′-Hydroxyl Acylation analyzed by Primer Extension) experiments. Originally developed as an Excel/VBA solution, the modernized Python version offers improved cross-platform compatibility, better signal alignment algorithms, and reduced manual intervention — while preserving the data transparency that researchers depend on.

  • Processes FSA capillary electrophoresis files
  • Automated peak detection and alignment
  • SHAPE reactivity normalization
  • Cross-platform (Windows, Mac, Linux)
  • Streamlit-based interactive interface
  • Open source and freely available

Cantara, W.A. et al. (2017). RiboCAT: A new capillary electrophoresis data analysis tool for nucleic acid probing. RNA. rnajournal.cshlp.org
